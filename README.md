# TeacherTI

TeacherTI 是一个面向教师群体的娱乐型人格测试 H5。它借用了人格测试的传播形式，但题面、维度和结果都落在老师熟悉的真实场景里，比如课堂气氛、教学路径、学生推动方式和成长策略。

当前版本为 24 道三选一题，完成后会生成四字母教师人格码、结果解读和一张可下载的分享卡。整个项目是纯前端静态站，直接打开 `index.html` 就能运行。

当前在线地址：

`https://agentshan.github.io/teacherti/`

## 项目简介

- 纯前端实现，部署成本低，适合做轻量传播和活动互动
- 包含题库、计分逻辑、16 型结果页和分享卡生成
- 结果页会展示人格标签、原型人物、倾向解读和日常建议

## 免责声明

本项目仅供娱乐和交流，不构成心理测评、职业评估、教育能力判断或任何正式结论。

请勿将结果用于绩效、评优、排课、招聘、晋升、心理诊断，或任何需要严肃判断依据的场景。
